# my-app — Route Studio demo project

Sample Next.js App Router project scanned by Route Studio. Each route demonstrates a different App Router concept.

| Route | Concepts |
|-------|----------|
| `/` | Root layout + page |
| `/dashboard` | Nested layout, `loading.tsx` |
| `/dashboard/settings` | `force-dynamic`, `fetch` + `no-store`, client boundary |
| `/login`, `/register` | Route group `(auth)` — omitted from URL |
| `/api/health` | Route handler |
| `proxy.ts` | Next.js 16 proxy (formerly middleware) |

Used by the live demo at `/studio` — no upload required yet.
